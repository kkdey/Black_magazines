Kushal K Dey

Introduction

In this script, we investigate the association between words or word sets of interest for the Black World/ Negro Digest magazine when the identity difference between black and negro terms has been nullified. We also track the associations across time.

library(wordVectors)
library(ggplot2)

Load word2vec models

We load the word2vec model fits.

dir <- "../BL_ND_ID/"
ll <- list.files(dir, pattern = ".bin")
model_list <- list()
for(m in 1:length(ll)){
  model_list[[m]] = read.vectors(paste0(dir, ll[m]))
}
